\# SecureERP — UI Baseline DevExpress NAV Style



\## Propósito

Definir el baseline de interfaz moderna estilo NAV / Dynamics usando DevExpress v25.



\## Estilo de UI

\- productiva

\- orientada a teclado y velocidad

\- lista + búsqueda + detalle

\- navegación lateral dinámica

\- acciones visibles según permisos

\- look \& feel moderno enterprise



\## Tecnología

\- WinForms

\- DevExpress v25



\## Patrones base de pantalla

1\. List Page

2\. Detail Page

3\. Search + Filters

4\. Action Bar / Ribbon / Toolbar

5\. Side navigation

6\. Context tabs si aplica



\## Navegación

Debe ser dinámica desde permisos efectivos y recursos UI.



\## Bloques transversales reutilizables

\- documentos

\- comentarios

\- etiquetas

\- historial resumido

\- estado / workflow

\- auditoría resumida si aplica



\## Acciones estándar posibles por recurso

\- agregar

\- editar

\- ver detalle

\- listar / buscar paginado

\- activar

\- desactivar

\- imprimir

\- exportar

\- adjuntar documento

\- ver documentos

\- comentar

\- ver comentarios

\- etiquetar

\- ver etiquetas

\- enviar a aprobación

\- aprobar

\- rechazar

\- anular

\- reversar



\## Reglas duras

\- no hardcodear permisos en UI

\- ocultar botones no reemplaza seguridad backend

\- la UI consume navegación y acciones permitidas desde API

\- componentes reutilizables primero, pantallas específicas después
